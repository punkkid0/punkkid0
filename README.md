# About Me

Full-Stack Developer & Ethical Hacker building production web/mobile apps, media tooling, WordPress sites, and Python security utilities.

I ship full-stack systems end-to-end — React frontends, Node.js/Express APIs, PostgreSQL, Firebase, Capacitor Android, and WordPress — plus **API integrations** that connect products to real services. On the security side I build recon/OSINT tooling, crypto labs, and practice on platforms like **TryHackMe**, **Hack The Box**, **PortSwigger Web Security Academy**, and **OWASP Juice Shop**, under the **ramseverywhere** brand.

My product work includes **Afrifit** (nutrition dashboard for Nigerian & African foods with Firebase + Android packaging), school **payment verification & clearance** systems (auto-verify receipts, JWT auth, PWA), and media tooling (ID3 tagging, sticker/video pipelines, upload validation).

On this GitHub you’ll find case studies of private products, public Python tools, CLI security utilities, and practical automation for labs and authorized testing.

I care about clean architecture, secure defaults, and tools that hold up in real workflows — apps people use, and tooling pentesters/builders actually run.

I’m especially interested in work where **software engineering meets cybersecurity** — secure full-stack products, API-driven systems, recon automation, and educational security tooling.
